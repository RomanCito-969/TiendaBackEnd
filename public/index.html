API RESTfull

ARMAS

GET /api/armas Obtener todos los arma.
POST /api/armas Añadir un arma.
GET /api/armas/{id} Obtener un arma.
UPDATE /api/armas/{id} Modificar un arma.
DELETE /api/armas/{id} Eliminar un arma.

CLIENTES

GET /api/clientes Obtener todos los clientes.
POST /api/clientes Añadir un cliente.
GET /api/clientes/{id} Obtener un cliente.
UPDATE /api/clientes/{id} Modificar un cliente.
DELETE /api/clientes/{id} Eliminar un cliente.
